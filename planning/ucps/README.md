# UCPs

This folder stores use-case planning pages and UCP generation parameters.

## Structure

- `parameters/` - versioned UCP generation parameter files.
- `active/` - official UCP pages for active milestones.
- `completed/YYYY-MM/` - official UCP pages for completed milestones.

## Rules

- UCP pages coordinate milestone sizing and use-case scope.
- Specs own design decisions.
- Plans own execution order, validation and closeout state.
- Completed UCP pages must move to `planning/ucps/completed/YYYY-MM/**`.
